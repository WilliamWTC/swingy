# swingy
implementation of a text-based RPG following the Model-View-Controller architecture and switching between the console view and GUI view.

# Compile & Run

$> mvn clean package

$> java -jar target/swingy-1.0.jar console / gui (Choose either console or gui)

# Result
Both console and gui views running.
